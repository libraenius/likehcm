# Варианты цветового оформления для Skillmap

Текущая схема: **Нейтральная (черно-белая)**

## Вариант 1: Синяя профессиональная тема
**Характеристики:** Корпоративная, надежная, профессиональная
**Подходит для:** Бизнес-приложений, корпоративных систем

```css
--primary: oklch(0.55 0.22 252); /* Синий */
--primary-foreground: oklch(0.98 0 0);
--accent: oklch(0.95 0.02 252); /* Светло-синий */
--accent-foreground: oklch(0.25 0.15 252);
```

## Вариант 2: Зеленая тема роста
**Характеристики:** Развитие, рост, прогресс, позитив
**Подходит для:** Систем управления навыками, образовательных платформ

```css
--primary: oklch(0.50 0.18 145); /* Зеленый */
--primary-foreground: oklch(0.98 0 0);
--accent: oklch(0.92 0.04 145); /* Светло-зеленый */
--accent-foreground: oklch(0.30 0.12 145);
```

## Вариант 3: Фиолетовая современная тема
**Характеристики:** Креативная, современная, инновационная
**Подходит для:** Современных веб-приложений, креативных платформ

```css
--primary: oklch(0.50 0.20 300); /* Фиолетовый */
--primary-foreground: oklch(0.98 0 0);
--accent: oklch(0.92 0.04 300); /* Светло-фиолетовый */
--accent-foreground: oklch(0.30 0.15 300);
```

## Вариант 4: Оранжевая энергичная тема
**Характеристики:** Энергичная, мотивирующая, активная
**Подходит для:** Мотивационных платформ, систем достижений

```css
--primary: oklch(0.65 0.20 45); /* Оранжевый */
--primary-foreground: oklch(0.15 0 0);
--accent: oklch(0.95 0.05 45); /* Светло-оранжевый */
--accent-foreground: oklch(0.40 0.15 45);
```

## Вариант 5: Бирюзовая/циановая тема
**Характеристики:** Свежая, технологичная, чистая
**Подходит для:** Технических платформ, систем аналитики

```css
--primary: oklch(0.55 0.18 200); /* Бирюзовый */
--primary-foreground: oklch(0.98 0 0);
--accent: oklch(0.92 0.04 200); /* Светло-бирюзовый */
--accent-foreground: oklch(0.30 0.12 200);
```

## Вариант 6: Индиго тема
**Характеристики:** Балансированная, спокойная, профессиональная
**Подходит для:** Универсальных приложений, систем управления

```css
--primary: oklch(0.45 0.18 270); /* Индиго */
--primary-foreground: oklch(0.98 0 0);
--accent: oklch(0.93 0.03 270); /* Светло-индиго */
--accent-foreground: oklch(0.28 0.14 270);
```

## Рекомендация для Skillmap
**Рекомендуется: Вариант 2 (Зеленая тема роста)**
- Символизирует развитие и рост навыков
- Ассоциируется с прогрессом и достижениями
- Подходит для образовательных и HR-платформ
- Приятна для глаз при длительной работе

## Как применить схему
1. Откройте файл `src/app/globals.css`
2. Найдите секцию `:root` или `.dark`
3. Замените значения `--primary`, `--accent` и связанных переменных
4. Сохраните и проверьте результат
