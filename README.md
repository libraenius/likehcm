# РИТМ — HCM-система

Внутренняя корпоративная система управления человеческим капиталом (Human Capital Management). Включает управление компетенциями, карьерными треками, оценочными процедурами и аналитику развития сотрудников.

## Модули системы

### Компетенции и профили
- Управление справочниками компетенций (профессиональные и корпоративные)
- Профили должностей с привязкой компетенций и уровней
- Карьерные треки с отслеживанием прогресса
- Самооценка навыков сотрудниками

### Окно Джохари — многоролевая оценка компетенций
Модуль оценки 360 на базе модели «Окно Джохари», позволяющий выявить разрыв между самовосприятием сотрудника и восприятием окружающих.

**Роли оценщиков:** самооценка, руководитель, коллеги, подчинённые.

**Визуализация четырёх квадрантов:**
- **Открытая зона** — компетенции, признаваемые и самим сотрудником, и окружающими
- **Слепая зона** — сильные стороны, которые видят другие, но не осознаёт сам сотрудник
- **Скрытая зона** — навыки, которые сотрудник знает за собой, но окружающие не замечают
- **Неизвестная зона** — потенциал для развития, пока не раскрытый ни одной из сторон

**Аналитика:** агрегация оценок, приоритизация зон развития, рекомендации по расширению открытой зоны, покрытие оценками по ролям.

### Оценки внешних провайдеров
Сервис загрузки и управления отчётами от внешних оценочных провайдеров (SHL, Hogan, Zetic, Azimut, HT Lab и др.) с иерархическим контролем доступа.

**Управление провайдерами и процедурами:**
- Создание провайдеров и оценочных процедур с датами, участниками, ссылками и QR-кодами
- Формирование списка участников из оргструктуры или импортом файла от провайдера
- Пакетная загрузка индивидуальных отчётов (ZIP-архив с PDF по ID сотрудников)
- Загрузка групповых, сводных, общих и аналитических отчётов с привязкой к процедуре

**Типы отчётов и видимость:**
- Индивидуальные, групповые, сводные, общие, аналитические отчёты
- Гибкая матрица видимости: по цепочке руководителей, по иерархии подразделений, для заказчиков исследования
- Персональный доступ по ФИО сотрудника для кросс-функциональных команд
- Предпросмотр доступа от разных ролей (сотрудник, начальник отдела, директор и т.д.)

**Дашборд руководителя — «Оценочные процедуры сотрудников»:**
- Сводная таблица подчинённых со всеми видами оценок: внешние процедуры, 360/ФКР, Ассессмент-центр, ЦОР
- Карточки внешних процедур с датами, статусами, типами отчётов и кнопками действий
- Модалки информации о процедуре и результатов с доступными отчётами
- Поиск по ФИО + модалка фильтров по подразделению и статусам
- Подача заявки на оценку руководителем (статус «заявка» → принятие/отклонение администратором)

**Администрирование:**
- Дерево провайдеров и процедур с поиском и фильтрацией по статусу
- Управление участниками, загрузка отчётов, настройка матрицы видимости
- Обработка заявок на оценку от руководителей
- Уведомления участникам процедур

### Прочие модули
- ИПР и 1-on-1 встречи
- Цели и КПЭ
- Кадровый резерв (сакцессия)
- Карьерные треки
- Стажировки и взаимодействие с ВУЗами

## Технологии

- **Next.js 16** — React-фреймворк с App Router
- **React 19** — UI-библиотека
- **TypeScript** — строгая типизация
- **Tailwind CSS** — утилитарная стилизация
- **Shadcn UI / Radix UI** — компонентная библиотека
- **Zod** — валидация данных
- **React Hook Form** — управление формами

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

### Запуск без терминала (Windows)

Дважды щёлкните по **`start-dev.bat`** в корне проекта — скрипт сам проверит Node.js и зависимости, при необходимости выполнит `npm install` и запустит dev-сервер на http://localhost:3000.

```bash
# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```
