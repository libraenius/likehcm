# Многосторонняя оценка (360°) и окно Джохари - Итоговый отчет

## Основные возможности

### Для сотрудников

1. **Самооценка компетенций**
   - Оцените свои компетенции по шкале 1-5
   - Добавьте комментарии к оценкам
   - Отслеживайте свой прогресс

2. **Сбор обратной связи**
   - Получайте оценки от руководителя
   - Собирайте анонимные оценки от коллег
   - Получайте оценки от подчиненных (для руководителей)

3. **Окно Джохари**
   - Визуализация сильных сторон (Arena)
   - Выявление слепых зон (Blind Spot)
   - Обнаружение переоценки (Façade)
   - Определение зон развития (Unknown)

4. **Аналитика**
   - Сравнение самооценки с оценками других
   - Выявление наибольших разрывов
   - Рекомендации по развитию

### Для руководителей

1. **Оценка подчиненных**
   - Оценка компетенций сотрудников
   - Добавление комментариев
   - Отслеживание развития

2. **360° обратная связь**
   - Комплексная оценка с разных точек зрения
   - Объективная картина компетенций
   - Основа для планирования развития

## Лучшие практики использования

### Сценарий 1: Регулярная оценка
```
1. Самооценка (15 минут)
2. Сбор оценок от руководителя (10 минут)
3. Сбор оценок от 3-5 коллег (по 10 минут каждому)
4. Анализ окна Джохари (20 минут)
5. Составление плана развития (30 минут)

Итого: ~2 часа на полный цикл оценки
Рекомендуемая частота: 1 раз в 6 месяцев
```

### Сценарий 2: Подготовка к аттестации
```
1. Проведите 360° оценку за 2 недели до аттестации
2. Изучите окно Джохари
3. Подготовьте примеры для сильных сторон
4. Составьте план развития для зон роста
5. Обсудите результаты с руководителем
```

### Сценарий 3: Адаптация нового сотрудника
```
Месяц 1: Самооценка (baseline)
Месяц 3: 360° оценка от команды
Месяц 6: Повторная оценка + сравнение прогресса
```

## Окно Джохари - Детально

### Квадранты

| Квадрант | Самооценка | Оценка других | Значение | Действие |
|----------|------------|---------------|----------|----------|
| **Arena** (Открытая область) 🟢 | Высокая (≥3.5) | Высокая (≥3.5) | Подтвержденные сильные стороны | Продолжайте развивать, делитесь опытом |
| **Blind Spot** (Слепая зона) 🔵 | Низкая (<3.5) | Высокая (≥3.5) | Недооценка способностей | Запросите примеры, повысьте уверенность |
| **Façade** (Скрытая область) 🟠 | Высокая (≥3.5) | Низкая (<3.5) | Переоценка или зона развития | Получите обратную связь, найдите ментора |
| **Unknown** (Неизвестная область) ⚫ | Низкая (<3.5) | Низкая (<3.5) | Зона для развития | Составьте план обучения |

### Интерпретация разрывов

- **+1.5 и выше**: Значительная переоценка - требуется коррекция восприятия
- **+0.5 до +1.5**: Умеренная переоценка - зона для развития
- **-0.5 до +0.5**: Согласованность - реалистичная оценка
- **-0.5 до -1.5**: Умеренная недооценка - повысьте уверенность
- **-1.5 и ниже**: Значительная недооценка - признайте свои достижения

## Конфиденциальность и анонимность

### Гарантии анонимности:
- ✅ Оценки коллег агрегируются (минимум 3)
- ✅ Оценки подчиненных агрегируются (минимум 2)
- ✅ Видна только средняя оценка, не индивидуальные
- ✅ Комментарии коллег анонимны

### Что видно оцениваемому:
- Своя самооценка (полностью)
- Оценка руководителя (с именем и комментарием)
- Средняя оценка коллег (без имен)
- Средняя оценка подчиненных (без имен)
