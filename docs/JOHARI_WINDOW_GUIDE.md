# Окно Джохари и многосторонняя оценка (360°)

## Обзор

Система многосторонней оценки компетенций с визуализацией через окно Джохари - психологический инструмент для понимания самовосприятия через сравнение самооценки с восприятием окружающих.

## Что такое окно Джохари?

Окно Джохари разделяет компетенции на 4 квадранта:

### 1. Открытая область (Arena) 🟢
- **Описание**: Высокая самооценка + высокая оценка других
- **Значение**: Ваши подтвержденные сильные стороны
- **Действие**: Продолжайте развивать, делитесь опытом

### 2. Слепая зона (Blind Spot) 🔵
- **Описание**: Низкая самооценка + высокая оценка других
- **Значение**: Вы недооцениваете свои способности
- **Действие**: Запросите примеры, повысьте уверенность

### 3. Скрытая область (Façade) 🟠
- **Описание**: Высокая самооценка + низкая оценка других
- **Значение**: Возможна переоценка или зона для развития
- **Действие**: Получите обратную связь, найдите ментора

### 4. Неизвестная область (Unknown) ⚫
- **Описание**: Низкая самооценка + низкая оценка других
- **Значение**: Зона для развития
- **Действие**: Составьте план обучения

## Роли в многосторонней оценке

### 1. Самооценка
- Основа для построения окна Джохари
- Оцените себя максимально объективно
- Ваши оценки видны только вам

### 2. Оценка руководителя
- Оценка на основе наблюдений за работой
- Видна сотруднику
- Вес в общей оценке: 40%

### 3. Оценка коллег
- Анонимная оценка
- Усредняется с оценками других коллег
- Вес в общей оценке: 40%
- Рекомендуется минимум 3 оценки

### 4. Оценка подчиненных
- Анонимная оценка
- Усредняется с оценками других подчиненных
- Вес в общей оценке: 20%

## Как использовать

### Шаг 1: Самооценка
1. Перейдите в раздел "Сервисы" → "Оценка"
2. Нажмите "Добавить" в карточке "Самооценка"
3. Оцените свои компетенции по шкале 1-5
4. Добавьте комментарии (необязательно)
5. Сохраните оценку

### Шаг 2: Сбор оценок от других
1. Попросите руководителя оценить ваши компетенции
2. Отправьте запросы коллегам (минимум 3)
3. Если есть подчиненные - получите их оценки

### Шаг 3: Просмотр окна Джохари
1. После получения минимум одной внешней оценки окно Джохари станет доступным
2. Изучите распределение компетенций по квадрантам
3. Обратите внимание на разрывы в оценках

### Шаг 4: Анализ и действия
1. Перейдите на вкладку "Аналитика"
2. Изучите детальную статистику
3. Определите приоритеты для развития
4. Составьте план действий на основе рекомендаций

## Лучшие практики

### Для самооценки
- ✅ Будьте честны с собой
- ✅ Приводите конкретные примеры в комментариях
- ✅ Оценивайте текущий уровень, а не желаемый
- ❌ Не завышайте оценки из-за синдрома самозванца
- ❌ Не занижайте из-за излишней скромности

### Для оценки других
- ✅ Основывайтесь на наблюдаемых фактах
- ✅ Оценивайте реальную работу, а не потенциал
- ✅ Добавляйте конструктивные комментарии
- ❌ Не оценивайте личность, оценивайте компетенции
- ❌ Не ставьте крайние оценки без обоснования

### Для получения качественных оценок
- ✅ Соберите минимум 3-5 оценок от коллег
- ✅ Выбирайте людей, которые видели вашу работу
- ✅ Обеспечьте анонимность оценок коллег
- ✅ Регулярно обновляйте оценки (раз в квартал/полгода)

## Интерпретация результатов

### Большой положительный разрыв (+1.5 и выше)
**Вы оцениваете себя выше, чем другие**
- Возможна переоценка своих способностей
- Запросите конкретные примеры улучшений
- Найдите ментора или пройдите обучение

### Большой отрицательный разрыв (-1.5 и ниже)
**Другие оценивают вас выше**
- Вы недооцениваете свои достижения
- Соберите примеры успешных кейсов
- Повысьте уверенность в своих силах

### Малый разрыв (от -0.5 до +0.5)
**Согласованность в оценках**
- Реалистичная самооценка
- Хорошее понимание своего уровня

## Сценарии использования

### Сценарий 1: Подготовка к аттестации
1. Проведите самооценку за 2 недели до аттестации
2. Соберите оценки от руководителя и коллег
3. Изучите окно Джохари
4. Подготовьте примеры для сильных сторон
5. Составьте план развития для зон роста

### Сценарий 2: Планирование карьеры
1. Определите целевую должность/профиль
2. Проведите многостороннюю оценку текущих компетенций
3. Сравните с требованиями целевой должности
4. Определите приоритетные компетенции для развития
5. Составьте ИПР (Индивидуальный План Развития)

### Сценарий 3: Адаптация в новой команде
1. Проведите самооценку через месяц работы
2. Через 3 месяца получите оценки от коллег
3. Выявите слепые зоны и скрытые сильные стороны
4. Скорректируйте коммуникацию и приоритеты

### Сценарий 4: Развитие команды (для руководителей)
1. Проведите 360° оценку всех членов команды
2. Проанализируйте общие паттерны
3. Выявите зоны роста команды
4. Организуйте обучение или обмен опытом
5. Повторите оценку через 6 месяцев

## Технические детали

### Структура данных

```typescript
interface MultiRoleAssessment {
  competenceId: string;
  selfAssessment?: SkillLevel; // 1-5
  managerAssessment?: SkillLevel;
  peerAssessments: SkillLevel[]; // Массив анонимных оценок
  subordinateAssessments: SkillLevel[];
  lastUpdated: Date;
  comments?: {
    self?: string;
    manager?: string;
    peers?: string[];
    subordinates?: string[];
  };
}
```

### Алгоритм определения квадранта

```typescript
// Порог между высокой и низкой оценкой
const threshold = 3.5;

const selfHigh = selfAssessment >= threshold;
const othersHigh = othersAverage >= threshold;

if (selfHigh && othersHigh) return "arena";      // Открытая область
if (!selfHigh && othersHigh) return "blind_spot"; // Слепая зона
if (selfHigh && !othersHigh) return "facade";     // Скрытая область
return "unknown";                                  // Неизвестная область
```

### Расчет средней оценки других

Взвешенное среднее:
- Руководитель: 40%
- Коллеги (средняя): 40%
- Подчиненные (средняя): 20%

## Конфиденциальность

### Что видно оцениваемому:
- ✅ Своя самооценка
- ✅ Оценка руководителя (с именем)
- ✅ Средняя оценка коллег (анонимно)
- ✅ Средняя оценка подчиненных (анонимно)
- ❌ Индивидуальные оценки коллег
- ❌ Индивидуальные оценки подчиненных

### Рекомендации по конфиденциальности:
- Минимум 3 оценки от коллег для анонимности
- Не обсуждайте конкретные оценки публично
- Храните результаты в безопасности

## Часто задаваемые вопросы

### Как часто обновлять оценки?
Рекомендуется раз в 6 месяцев или при смене проекта/роли.

### Что делать, если оценки сильно расходятся?
1. Запросите конкретные примеры
2. Обсудите с руководителем или HR
3. Проведите дополнительную сессию обратной связи

### Минимальное количество оценок для надежности?
- Самооценка: обязательна
- Руководитель: желательна
- Коллеги: минимум 3
- Подчиненные: минимум 2

### Можно ли изменить оценку после сохранения?
Да, оценки можно обновлять в любое время. Сохраняется последняя версия.
